Deskripsi    : Program ini merupakan program sederhana yang mensimulasikan
               bagaimana suatu mesin ATM bekerja, ATM ini memiliki beberapa
               fitur atau menu, diantaranya top-up pulsa, setor tunai,
	       tarik tunai, info saldo, ganti pin, dan info kurs. Kami
	       memanfaatkan file .txt untuk menyimpan data nasabah berupa nomor
	       rekening, pin, dan saldo nasabah. File program bisa diletakkan di
	       drive manapun.

Perbaikan    : Pin sudah dimasking dengan karakter lain, baik saat akan
	       memasukkan pin pertama kali atau melakukan ganti pin.
